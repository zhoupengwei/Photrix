Photrix is a modern, high-precision library for 3D reconstruction and photogrammetry, designed for researchers and developers in computer vision, geospatial analysis, and photogrammetric applications.

**Code release preparation in progress.**

## Features

* High-precision 3D reconstruction from images and point clouds

* Photogrammetric view-graph selection and optimization

* Multi-platform support (Linux / Windows)

* Supports C++20 and C++23 projects

* Optional CUDA acceleration (12.1â€“12.8)

* Python bindings for scripting and prototyping

* CMake-based build system for easy integration
